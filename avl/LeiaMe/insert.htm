Função de Inserção Balanceada

Inicializa a arvore caso ela for nula, insere de forma recursiva:
- A esquerda se a nova chave for menor que a chave atual
- A direita se a nova chave for maior que a chave atual
Chama as funcoes de rotacao quando necessario

Parâmetros:

AVLTree **node Ponteiro duplo que aponta para o nó raiz e recursivamente, cada nó que será acessado.
int value Inteiro que é a chave que está sendo inserida.
int *changeHeight Ponteiro de inteiro para verificar se a altura da árvore foi modifacada a cada inserção

Uso por:

main(...); Direciona o usuário a usar todas as funções disponíveis no programa, coleta os valores que serão inseridos na árvore.

Usa:

rotationRight(...); Executa a rotação à direita quando é encontrado um erro no balanceamento da árvore;
Faz a rotacao simples ou dupla, na mesma funcao;
a rotacao simples e dupla ocorrem de forma totalmente desconexas.
rotationLeft(...); Executa a rotacao a esquerda quando e encontrado um erro no balanceamento da arvore;
Faz a rotacao simples ou dupla, na mesma funcao;
a rotacao simples e dupla ocorrem de forma totalmente desconexas.

Retorno:

intInteiro, usado como booleano para definir sucesso ou inssucesso da operação de inserção.
